# Benchmark Datasets

1.  **Set A (`data/SetA/`)**:
    These instances were generated and introduced by:
    > Guignard M, Hahn PM, Pessoa AA, da Silva DC (2012). Algorithms for the cross-dock door assignment problem. *Proceedings of the fourth international workshop on model-based metaheuristics*, 145–162.

2.  **Set B (`data/SetB/`)**:
    These instances were generated and introduced by:
    > Nassief W, Contreras I, As’ Ad R (2016). A mixed-integer programming formulation and lagrangean relaxation for the cross-dock door assignment problem. *International Journal of Production Research*, 54(2):494–508.
